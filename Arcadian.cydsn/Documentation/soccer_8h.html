RC_t SOCCER_cyclic ( )

Function for the cyclic execution of the soccer game. Generates events and puts them in the ring buffer.

Returns
RC_t - RC_SUCCESS if successful.

The main cyclic function called while playing the game. Checks for button presses. Handles game events based on button presses. Put events into the ring buffer. Calls the control function.

Returns
RC_t - RC_SUCCESS if everything works fine
