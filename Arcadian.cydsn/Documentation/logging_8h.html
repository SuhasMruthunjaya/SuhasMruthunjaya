◆ logging_level_t

enum logging_level_t

Debugging Level.

The debugging level defines the character of an event and is used to define which events are send to the log using #LOGGING_LEVEL

Enumerator
LOG_VERBOSE 

Low Level Information.

LOG_INFO 

High level information during normal operation.

LOG_WARNING 

abnormal operation, which might be fixed

LOG_ERROR 

abnormal operation, which can not be fixed

LOG_DEBUG 

high priority for debugging messages (use only temporarily for debugging)

LOG_OFF 

Event logging is switched off.
