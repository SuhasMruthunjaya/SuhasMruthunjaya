RC_t LOG_send ( logging_t *  entry)

Send a new logging message to the logging-queue. It is recommended to use #LOGGING_SEND instead of this function directly.

Parameters
entryLogging message, see logging_t.
Returns
RC_ERROR_INVALID_STATE if the logging service is not available. RC_SUCCESS if the message was added to the queue. RC_ERROR_BUFFER_FULL if the queue is full and the message was not added.
