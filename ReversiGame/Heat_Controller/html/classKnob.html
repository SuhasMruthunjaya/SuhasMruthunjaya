Detailed Description

User Input Class.

Importing the VoltageRegulator.h header files is necessary as we need to get the Regulated voltage via the Voltage Regulator class. Since the Voltage recieved from the Power Supply Component is usually very high than the desired value. Keyboard or a knob regulator to set the desired room Temperature. The desired_value is inputed by the User.

void Knob::get_Decoder ( float *  user_input_value)

This function is used to take inputs from the User, the user turns the knob, which is attached to a incremental decoder, this function wraps the decoder driver, which gives the value of the pulses rotated, this is then used to deduce how much degree the user wants to increase / reduce the current temperature.
