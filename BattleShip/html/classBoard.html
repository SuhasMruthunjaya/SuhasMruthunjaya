Detailed Description

Represents a game board consisting of an OwnGrid and an OpponentGrid.

The Board class serves as the container for both the player's own grid and the opponent's grid in a game. It provides access to grid-related functionalities and dimensions.

Note
The board is initialized with specified rows and columns for the OwnGrid and OpponentGrid.
